  • ¿Qué es BitTorrent?

    • BitTorrent es un protocolo (un conjunto de normas y la descripción de cómo hacer las cosas) que te permite descargar archivos rápidamente, al mismo tiempo que compartis con otras personas partes del archivo que se esta descargando. BitTorrent se utiliza a menudo para la distribución de archivos muy grandes o archivos muy populares de distribución gratuita, ya que es mucho más barato, más rápido y eficaz de distribuir archivos con BitTorrent, que una descarga normal.

  • ¿Dónde consigo archivos .torrent?

    • Al contrario que otras redes de intercambio, BitTorrent no incluye ningún mecanismo de búsqueda de archivos. Los usuarios de BitTorrent deberán localizar por sus propios medios los archivos torrent que necesita el protocolo. Normalmente, estos archivos pueden descargarse desde las páginas web que publican grandes archivos (como las distribuciones GNU/Linux) o desde índices web de búsqueda (como The Pirate Bay, Ktorrents o Bitgle).
